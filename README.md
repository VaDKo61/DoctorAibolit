# Doctor Aibolit

Напоминает пациентам Aibolita о приёме лекарств.

# Установка

1. Клонируйте репозиторий

        https://github.com/VaDKo61/DoctorAibolit
        cd doctor_aibolit

2. Установите зависимости

        pip install -r requirements.txt

3. Запустите сервер

        python manage.py runserver
